Analisis de datos oraganizados

Este es un proyecto de análisis de datos, trabaja con datos ordenados de una encuesta de 2022 de usuarios de StackOverFlow y realiza una variedad de operaciones de análisis y comparación de datos usando Python y bibliotecas como NumPy, Pandas, PyPlot y Math.

Algoritmo genetico para detectar circulos

Este programa implementa un algoritmo genético para procesar imágenes y encontrar círculos en una imagen en escala de grises. Define una estructura INDIVIDUO que representa a cada individuo en la población, con cromosomas, valores decodificados, fitness y atributos del círculo (centro y radio). El programa también incluye funciones para evaluar la aptitud de los individuos y evolucionar la población a través de generaciones. El objetivo es ajustar los parámetros del círculo para que se superpongan con los píxeles negros en la imagen.
